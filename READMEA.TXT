## 📖 About The Project

A full-stack **To-Do List Web Application** built with Flask and SQLite. This project allows users to create, manage, and track their daily tasks with a clean and intuitive interface. Perfect for personal productivity or as a learning project for Flask and database operations.

## ✨ Features

### 📋 Task Management
- **Add Tasks** - Create new tasks with descriptions
- **View Tasks** - See all tasks in an organized list
- **Mark as Complete** - Track completed tasks
- **Delete Tasks** - Remove tasks you no longer need
- **Task Status** - Visual distinction between pending and completed tasks

### 🎨 User Interface
- Clean and responsive design
- Task counter showing pending tasks
- Easy-to-use forms for task entry
- Visual feedback for actions

### 🗄️ Database
- SQLite database for persistent storage
- Automatic database creation
- Simple data model for tasks
